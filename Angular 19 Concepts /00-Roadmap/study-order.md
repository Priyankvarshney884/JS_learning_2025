# Angular 19 Study Order

## Week 1 - Foundations

1. Angular architecture and project structure
2. Standalone components
3. Template syntax (`@if`, `@for`, interpolation, binding)
4. Signals basics (`signal`, `computed`, `effect`)

## Week 2 - Core App Building

1. Component communication (`input`, `output`, shared service)
2. Dependency Injection (`inject`, provider scopes)
3. Lifecycle hooks (only practical ones)
4. Pipes and directives (custom + built-in)

## Week 3 - Real App Features

1. Routing (lazy loading, route params, guards)
2. Forms (template-driven vs reactive)
3. HTTP + RxJS patterns (`switchMap`, `catchError`, retry/cancel)
4. Error handling and interceptors

## Week 4 - Interview + Production

1. Change detection and performance strategy
2. State handling options (signals store/service patterns)
3. Unit testing basics (component, service, HTTP)
4. Build, deploy, and architecture discussion

## Daily Rule

- 1 concept note
- 1 proof artifact
- 1 recall session (5-10 min)
